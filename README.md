# Word Counter Website

This is a simple web application that counts the number of words in a given text. It provides a user-friendly interface where users can enter their text and instantly see the word count after clicking on count button.

<hr>
## Demo

You can try the live demo of the Word Counter Website [here]([https://amit-kumar-pandey-05.github.io/Word-Counter/]).
- Link: [Live Demo](https://amit-kumar-pandey-05.github.io/Word-Counter/) | [GitHub Repo](https://github.com/Amit-Kumar-Pandey-05/Word-Counter)

<hr>
## Features

- Counts the number of words in a given text
- User-friendly interface
- Instant word count update

## Getting Started

<h3>To get a local copy up and running, follow these simple steps:</h3>

1. Clone the repository:
   git clone https://github.com/Amit-Kumar-Pandey-05/Word-Counter.git
   
2. Navigate to the project directory:
  cd word-counter-website

3. Open the "index.html" file in your preferred browser.

<hr>
<h3>How It Works</h3>
  Enter your text in the provided text area.
  Click the "Count Words" button.
  The website will count the number of words in the text and display the result below the button.
